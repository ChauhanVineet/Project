## How to Use

1. **Open `index.html` in your browser**
2. **Create a Workout:**
   - Go to "Workout" tab
   - Enter workout name
   - Add exercises with sets, reps, and rest time
   - Save workout

3. **Start Training:**
   - Go to "Home" tab
   - Click "Start Workout"
   - Follow voice guidance
   - Log weight and reps for each set
   - Rest timer starts automatically

4. **Track Progress:**
   - Go to "Progress" tab
   - Log body measurements
   - View workout history

5. **Track Nutrition:**
   - Go to "Nutrition" tab
   - Log meals with macros
   - Monitor daily intake

## Installation as PWA

### Desktop (Chrome/Edge):
1. Open the app in browser
2. Click the install icon in the address bar
3. Click "Install"

### Mobile (Android):
1. Open in Chrome
2. Tap the menu (3 dots)
3. Tap "Add to Home Screen"

### Mobile (iOS):
1. Open in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
